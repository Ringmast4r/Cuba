Servicios

Desarrollo de Software

Desarrollo de sitios web y portales. Software de gestión. Desarrollo de aplicaciones para móviles, juegos, ebooks y audiolibros.

Registro nacional de nombres de dominio

CITMATEL administra el Centro Cubano de Información de Red (CUBANIC), para el otorgamiento e inscripción en el DNS de los nombres de dominio del indicativo del país con plataformas propias.

Hospedaje de sitios Web

Hospedaje en su Centro de datos que garantiza el acceso a las estadísticas y el servicio los 365 días del año.

Hospedaje de servidores

Este servicio le permite al usuario tener su propio servidor de Internet garantizándole independencia y grandes facilidades. Sus bases de datos, aplicaciones y páginas Web corren en una página exclusiva que les aportará gran rapidez y visibilidad...

Servicios técnicos a sistemas de cómputo

Se ofrecen servicios de venta de equipamiento informático, mantenimiento y reparación de equipos, ensamblaje de computadoras, instalación de sistemas operativos y garantía postventa. Todo con atención personalizada y compromiso de calidad.

Cartelería Digital

Servicio de videoconferencia

Servicio de Impresión 3D

Instalación de Sistemas Fotovoltaicos
